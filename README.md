## Getting Started

Download the repo outside your .config directory
```bash
git clone https://github.com/ch3ber/dotfiles.git
cd dotfiles
```

Install the dependencies

- Bspwm
- Alacritty
- neovim
- plugged (nvim plugin manager)
- picom
- sxhkd

or use a script for arch linux
```bash
./dependencies.sh
```
